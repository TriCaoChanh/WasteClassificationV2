# Waste Classification Version 2
Training and deploying an AI Waste Classifier for Professional Skill for Engineering Project at HCMUT.
- **Update from version 1:** From 2 types *(Recyclable and Organic)* to 7 types *(Carboard, Glass, Metal, Organic, Paper, Plastic, and Trash)*

## Try it out:
- Streamlit App: [Waste Classification Web App](https://tricaochanh-wasteclassificationv2-app-ikftco.streamlit.app)
- Dataset used: [Waste Classification data](https://www.kaggle.com/datasets/techsash/waste-classification-data)
